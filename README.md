# Employee MIS Dashboard (Excel)

## Overview
This project is an Employee MIS Dashboard created in Microsoft Excel.

## Features
- KPI cards (Total, Active, Resigned, On Hold)
- Department-wise employee status
- Salary range distribution
- Manager-wise employee count
- Dynamic pivot tables and charts

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Slicers

## Use Case
Helps management understand workforce status, salary distribution and key trends quickly.
